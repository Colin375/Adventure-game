# Running Adventure-game
To run this file you first need a valid php installation which can be run in cmd, then in the php command prompt you just run the file by typing "php adventure-game.php" if in the same folder, otherwise change adventure-game.php to be the entire path including file name to the file you are trying to run.

# What is this?
This project is a short game with multiple endings which can only be played via a terminal.
